===============================================================================
                    E-COMMERCE WEB APPLICATION
===============================================================================

Project Type
------------
Multi-Vendor E-Commerce Web Application

Tech Stack
----------
Backend  : Django + Django REST Framework
Frontend : React (Vite)
Database : SQLite (Development) + MySQL (Production)
Authentication : JWT

===============================================================================
                               PROJECT GOAL
===============================================================================

Build a complete Multi-Vendor E-Commerce Web Application using Django REST
Framework as the backend and React as the frontend with JWT authentication,
role-based access control, product management, shopping cart, payment gateway,
dashboards, analytics, and deployment-ready architecture.
