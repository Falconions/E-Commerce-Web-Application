-------------------------------------------------------------------------------
			   Tools You'll Need
-------------------------------------------------------------------------------

Backend
-------

| Tool                          | Purpose                                |
| ----------------------------- | -------------------------------------- |
| Python 3.12+                  | Programming language                   |
| Django                        | Backend framework                      |
| Django REST Framework         | Build REST APIs                        |
| MySQL	                        | Production database                    |
| SQLite                        | Development database                   |
| Pillow                        | Image uploads                          |
| django-filter                 | Search and filtering                   |
| djangorestframework-simplejwt | JWT Authentication                     |
| django-cors-headers           | Allow React to communicate with Django |


Frontend
--------

| Tool                      | Purpose                  |
| ------------------------- | ------------------------ |
| React                     | User Interface           |
| Vite                      | Fast React project setup |
| React Router              | Page routing             |
| Axios                     | API requests             |
| Bootstrap or Tailwind CSS | Styling                  |
| React Hook Form           | Form handling            |
| Chart.js or Recharts      | Dashboard charts         |
